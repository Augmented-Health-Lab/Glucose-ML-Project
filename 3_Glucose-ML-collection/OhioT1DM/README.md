# OhioT1DM: Glucose-ML-collection Dataset

This is a controlled-access dataset. Existing data use agreements (DUA) do not permit access to the extracted glucose files and metadata for this dataset. Please request access at https://pmc.ncbi.nlm.nih.gov/articles/PMC7881904/, after which you can use our provided harmonize-cgm-dataset [add link] script for processing.
